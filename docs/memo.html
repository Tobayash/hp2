階層的重回帰分析

　階層的重回帰分析は、ステップ１からステップ２へというようにステップごとに関心のある変数を投入していき、分散説明率が統計的に有意に増加することを検定することで、その変数の重要性を分析する手法である。
　たとえば、チョコレートの摂取量に影響する要因として、甘いもの好きかとストレスが考えられるが、研究として関心があるのはこれらの交互作用効果だとする。その場合は、ステップ１では説明変数として“甘いの好きな程度”と“ストレス”を入れ、ステップ２ではさらに“甘いの好きな程度”と“ストレス”の交互作用項を入れて分析する。ただし、各変数と交互作用項の相関による多重共線性の問題が考えられるので中心化処理（平均値を引く）を忘れずに行う。
　Rで分析する場合には以下のコードを用いる。説明変数の数や共変量の有無は適宜調整する。

data<-XXX #分析したいデータを入れる
y<-data$v1  #応答変数
x1<-data$v2-mean(data$v2)  #説明変数（中心化）
x2<-data$v3-mean(data$v3)　#説明変数（中心化）
c1<-data$v4-mean(data$v4)　#共変量（中心化）

#第1ステップ
step1<-lm(y~x1+x2+c1)
summary(step1)

#第2ステップ
step2<-lm(y~x1+x2+x1:x2+c1)
summary(step2)

#平方和の変化量の検定
anova(step1,step2)

結果の書き方の例
チョコ得点を応答変数とした階層的重回帰分析を行った。その結果、ステップ1（R2adj = .24, F (3,71) = 6.23, p < .05）から交互作用項を含んだステップ2（R2adj = .31, F (4,70) = 7.01, p < .05）で分散説明率の有意な増加が見られた（ΔR2 = .07, ΔF (1,70) = 4.23, p < .05）。甘党得点とストレス得点の交互作用効果が有意であった（b* = 0.22, t (70) = 2.82, p < .05）。甘党得点の主効果（b* = 0.02, t (70) = 0.01, ns.）とストレス得点の主効果（b* = 0.15, t (70) = 1.30, ns.）は有意ではなかった。共変量の食事量の主効果が有意であった（b* = -2.11, t (70) = 4.12, p < .05）。

単純傾斜分析
交互作用効果が有意であった場合、下位検定として単純傾斜分析を行う。

#下位検定
## x1が+1SDの場合のx2の効果を分析
x1high<-data$v2-(mean(data$v2)+sd(data$v2))
simpleslope1<-lm(y~x1high+x2+x1high:x2+c1) 
summary(simpleslope1)

## x1が-1SDの場合のx2の効果を分析
x1low<-data$v2-(mean(data$v2)-sd(data$v2))
simpleslope2<-lm(y~x1low+x2+x1low:x2+c1)
summary(simpleslope2)

結果の書き方の例
甘党得点とストレス得点の交互作用効果が有意であったため、下位検定として単純傾斜分析を行った。その結果、甘党を-1SDとした場合、ストレス得点の有意な効果は認められなかった（b* = -0.15, t (70) = 0.46, ns.）。甘党得点を+1SDとした場合、ストレス得点の正の効果が確認された（b* = 2.33, t (70) = 3.21, p < .05）。

軸の設定

  • ggplotでグラフを作成すると、x軸から棒グラフなんかが浮いてしまう（ラピュタ）。ラピュタを無くすならscale_y_continuous(expand = c(0,0))を加える
  • グラフの縦軸の指定は、たとえば０～１０にしたかったらcoord_cartesian(ylim=c(0,10))
